# npiperelay Rust Implementation Documentation

This directory contains documentation for the Rust implementation of npiperelay, a tool for accessing Windows named pipes from WSL.

## Documents

- [overview.md](overview.md) - Project purpose, use cases, and architecture
- [cli.md](cli.md) - Complete command-line interface specification
- [data-flow.md](data-flow.md) - Core functionality and data flow details
- [windows-api.md](windows-api.md) - Windows named pipe handling and API usage
- [error-handling.md](error-handling.md) - Error handling behavior and edge cases
- [implementation-plan.md](implementation-plan.md) - Rust implementation plan and guidance
